### E005: CORS zu offen konfiguriert
**Datei:** `api.py:40-46`  
**Kategorie:** Security  
**Schwere:** HIGH  
**Impact:** CSRF-Angriffe m√∂glich

```python

    CORSMiddleware,
    allow_origins=allowed_origins,
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

**Problem:**
- `allow_methods=["*"]` erlaubt alle HTTP-Methoden

- Default-Origins enthalten localhost (unsicher in Produktion)

**Fix:**
```python

    CORSMiddleware,
    allow_origins=allowed_origins,  # Nur spezifische Domains
    allow_credentials=True,
    allow_methods=["GET", "POST", "PUT", "DELETE"],  # Spezifisch
    allow_headers=["Authorization", "Content-Type"],  # Spezifisch
)
```
